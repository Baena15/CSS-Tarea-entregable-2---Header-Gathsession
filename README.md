# GathSession - Header Hero

Proyecto de CSS desarrollando un header-hero para la landing page de GathSession, una plataforma para construir comunidades.

## 📋 Descripción

Este proyecto consiste en la maquetación de un header-hero estático basado en un diseño de Figma. Incluye navegación, sección hero con contenido textual, botones de acción, features y un grid de imágenes decorativas.

## 🚀 Demo

Puedes ver el proyecto en vivo aquí: [Enlace a GitHub Pages]

## 🛠️ Tecnologías utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos y diseño visual
- **SVG** - Iconos vectoriales escalables
- **Google Fonts** - Tipografías (Merriweather, Poppins)

## ✨ Características

- **Diseño estático** fiel al mockup de Figma
- **HTML semántico** con etiquetas apropiadas (header, nav, section, article)
- **Iconos SVG** para mejor calidad y escalabilidad
- **Estados hover** en botones y enlaces interactivos
- **CSS Variables** para mantenimiento de colores
- **Grid de imágenes** posicionado con CSS absolute

## 🔧 Instalación y uso

1. Clona el repositorio:
```bash
git clone [url-del-repositorio]
```

2. Navega al directorio:
```bash
cd dist
```

3. Abre el archivo `index.html` en tu navegador preferido.

O simplemente visita el enlace de GitHub Pages.

## 📱 Notas

- Este es un diseño **estático** optimizado para desktop (1280px+)
- No es responsive (según requerimientos del proyecto)
- Compatible con navegadores modernos (Chrome, Firefox, Safari, Edge)
